# 69hockey — Описание сайта

## О сайте
Многостраничный сайт 69hockey. Сайт представляет продукты компании в сфере хоккея.

---

## Навигация (хедер)
- **Турниры** → `/tournaments`
- **Кэмпы** → `/camps`
- **Партнёрство** → `/partnership`

---

## Страницы

### Главная (`/`)

#### Hero-секция
- Полноэкранный блок, первое что видит пользователь
- Заголовок: название или слоган сайта
- Подзаголовок: короткое описание
- CTA-кнопка (например "Смотреть турниры")

---

### Турниры (`/tournaments`)
Страница со списком всех турниров.

Каждый турнир отображается как карточка:
- Название турнира
- Даты проведения
- Место проведения
- Краткое описание
- Ссылка на страницу турнира

---

### Страница турнира (`/tournaments/[slug]`)
Отдельный лендинг для каждого турнира.

Содержимое:
- Название турнира
- Даты и место проведения
- Полное описание турнира
- Формат и регламент
- Призовой фонд / награды
- Расписание
- Блок регистрации / CTA

---

### Кэмпы (`/camps`)
Страница с хоккейными кэмпами. Структура аналогична турнирам — список кэмпов, у каждого своя страница.

---

### Партнёрство (`/partnership`)
Страница для потенциальных партнёров.

Содержимое:
- Описание возможностей для партнёров
- Форматы сотрудничества
- Контактная форма или CTA
